# MERN Stack Multi-Vendor E-Commerce Project

This is a full-stack e-commerce application developed as part of my learning journey in Information Technology. The project focuses on building a scalable retail platform with a secure backend and a responsive frontend.

## Key Features
* **Full-Stack Integration:** Built using MongoDB, Express.js, React.js, and Node.js (MERN).
* **MVC Architecture:** Organized using Model-View-Controller patterns for clean and maintainable code.
* **Authentication & Security:** Implemented secure user login and registration using JSON Web Tokens (JWT) and BCrypt encryption.
* **State Management:** Utilizes Redux for handling complex application states like cart and user data.
* **Responsive Design:** Styled with Tailwind CSS to ensure the store works perfectly on mobile and desktop.

## Tech Stack
* **Frontend:** React, Redux, Tailwind CSS.
* **Backend:** Node.js, Express.js.
* **Database:** MongoDB with Mongoose.
* **Tools:** Git, GitHub, VS Code, Vim.

## Why I Built This
As a BSc IT graduate, I developed this project to gain hands-on experience with real-world software development challenges, specifically focusing on authorization flows and database relationship management.
